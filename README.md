# Argo-for-Sac

* 新建私库，上传此项目里的index.js和package.json，本地修改好start.sh里需要修改的变量。

* 然后打开任意一个公开项目，点击右边的Releases，新建tag，下方上传start.sh,待上传完成，点击下方的publish确定上传
